    Download Terraform

    Below are the available downloads for the latest version of Terraform (0.13.5). Please download the proper package for your operating system and architecture.

    Terraform is distributed as a single binary. Install Terraform by unzipping it and moving it to a directory included in your system's PATH .

    You can find the SHA256 checksums for Terraform 0.13.5 online and you can verify the checksums signature file which has been signed using HashiCorp's GPG key. You can also download older versions of Terraform from the releases service.

    Check out the v0.13.5 CHANGELOG for information on the latest release.

    Note: When you upgrade to Terraform 0.13, your existing Terraform configurations might need syntax updates. You can make most of these updates automatically with the terraform 0.13upgrade command; for more information, see Upgrading to Terraform 0.13.
